# GROK-BOT  
### Смачно навайбкоженный ботик на питончике

> [!NOTE]
> это шуточная игрушка которая нужна исключительно для одного чата с моими друзьями

> [!TIP]
> Не используйте эту шнягу и даже не смотрите

> [!IMPORTANT]
> я выкладываю это на гитхаб потому что хостинг который я нашёл не может по-другому

> [!WARNING]
> Бот может послать вас куда подальше

> [!CAUTION]
> Ни в коем случчае не используйте эту фигню

# Использование
1. добавьте бота @grok_monkey_bot в чат и айте ему права администора(он не будет закрыавть вашу группу, хотя это интересная фича)
2. пропишите `/grokyesno` чтобы увидеть статисттику кто попался на прикол с да/нет
3. бот реагирует только на сообщения в которых есть слово `грок` и только по-русски (в планах доавить английский вариант) - отвечает да/нет/хамит
4. бот так же реагирует на ответы на его сообщения - посылает вас куда подальше
5. иногда ставит случайную реакцию - вероятность можно настроить но сейчас кажется 10%
6. и с какой-то вероятность отправляет в чат сообщение "Однажды один мудрец/глупец сказал:..." и дополняет это случайной строкой из чата. НО для работы этой функции надо сначала экспортировать чат из тг в формате json, запусть файл `messages-parser.py` который собственно распарсит ваш json в обычные строки и дальше можете запускать бота. строки в файл добавляются автоматически если бот видит сообщение

Планируется ли поддержка в будущем? - возможно буквально несколкьо доработок а так в перспективе ничего не будет
